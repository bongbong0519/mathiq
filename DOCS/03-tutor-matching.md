# 과외 매칭 시스템

> 최종 업데이트: 2026-04-21  
> 상태: ✅ Phase 1 완료

---

## 목적

학부모/학생과 과외 선생님을 연결해주는 매칭 플랫폼입니다.  
신뢰성 확보를 위해 포인트 기반 참여 방식을 사용합니다.

---

## 사용자

| 역할 | 할 수 있는 것 |
|------|---------------|
| 선생님 | 과외 프로필 등록, 매칭 신청 수락/거절 |
| 학부모/학생 | 구인 프로필 등록, 선생님 검색, 매칭 신청, 성사 신고 |

---

## 매칭 플로우

```
[학부모]                           [선생님]
    │                                  │
    │  1. 선생님 검색                   │
    │────────────────────────────────>│
    │                                  │
    │  2. 매칭 신청 (무료)              │
    │  + 신청 메시지 작성               │
    │────────────────────────────────>│
    │                                  │
    │                    3. 신청 검토   │
    │                    (구인 정보 참고)│
    │                                  │
    │  4-A. 수락 (500P 차감)           │
    │<────────────────────────────────│
    │  → 양쪽 연락처 공개               │
    │                                  │
    │  4-B. 거절 (비용 없음)            │
    │<────────────────────────────────│
    │                                  │
    │  5. 외부 연락 후 과외 시작         │
    │<─────────────────────────────────>│
    │                                  │
    │  6. 성사 신고 (500P 리워드)       │
    │────────────────────────────────>│
```

## 포인트 정책

| 행동 | 포인트 | 대상 |
|------|:------:|------|
| 매칭 신청 | 무료 | 학부모 |
| 매칭 수락 | -500P | 선생님 |
| 매칭 거절 | 무료 | 선생님 |
| 성사 신고 | +500P | 학부모 |

### 설계 의도
- **신청 무료**: 학부모 진입 장벽 낮춤
- **수락 시 차감**: 선생님의 진지한 참여 유도, 허위 수락 방지
- **성사 리워드**: 학부모의 성사 신고 유도, 실제 매칭률 파악

## 기능 상세

### 선생님 측

#### 과외 프로필 등록
- 과목 (복수 선택): 수학, 영어, 국어 등
- 지역 (복수 선택): 시/도 + 구/군
- 경력, 자기소개, 희망 시급
- 프로필 활성화/비활성화 토글

#### 매칭 관리 (📨 매칭 관리)
- **받은 신청** 탭: pending 상태 신청 목록
  - 학부모 이름, 신청 메시지, 구인 정보 확인
  - 수락 / 거절 버튼
- **수락한 매칭** 탭: 수락 완료된 매칭
  - 학부모 연락처(이메일, 전화) 확인
  - 복사 버튼
- **거절한 신청** 탭: 거절 기록

### 학부모/학생 측

#### 구인 프로필 등록
- 과목 (복수 선택)
- 지역 (복수 선택)
- 학년, 요청사항
- 프로필 활성화/비활성화

#### 선생님 찾기
- 과목/지역 필터 검색
- 선생님 카드 목록
- 상세 정보 패널 (클릭 시)
- "📨 매칭 신청하기" 버튼

#### 매칭 신청 모달
- 신청 메시지 작성 (10~500자)
- 안내 사항 표시
- 구인 정보 등록 링크 제공

#### 내 매칭 신청 (📨 내 매칭 신청)
- **대기 중** 탭: pending 신청
  - 취소 버튼
- **수락된 매칭** 탭: 수락된 매칭
  - 선생님 연락처 확인
  - "성사 신고" 버튼 (500P 리워드)
- **거절된 신청** 탭: 거절/취소 기록

## 데이터베이스

### tutor_profiles (선생님 프로필)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| user_id | UUID | PK, FK → profiles |
| subject | TEXT[] | 과목 배열 |
| region | TEXT[] | 지역 배열 |
| experience | TEXT | 경력 |
| introduction | TEXT | 자기소개 |
| hourly_rate | INTEGER | 희망 시급 |
| is_active | BOOLEAN | 활성화 여부 |

### tutee_profiles (구인 프로필)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| user_id | UUID | PK, FK → profiles |
| subject | TEXT[] | 과목 배열 |
| region | TEXT[] | 지역 배열 |
| grade | TEXT | 학년 |
| request | TEXT | 요청사항 |
| is_active | BOOLEAN | 활성화 여부 |

### tutor_match_requests (매칭 신청)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | UUID | PK |
| requester_id | UUID | 신청자 (학부모) |
| tutor_id | UUID | 선생님 |
| message | TEXT | 신청 메시지 |
| status | TEXT | pending/accepted/rejected/cancelled |
| created_at | TIMESTAMPTZ | 신청 시간 |
| responded_at | TIMESTAMPTZ | 응답 시간 |

### tutor_matches (성사된 매칭)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | UUID | PK |
| request_id | UUID | FK → tutor_match_requests |
| tutor_id | UUID | 선생님 |
| requester_id | UUID | 학부모 |
| confirmed_at | TIMESTAMPTZ | 수락 시점 (500P 차감) |
| reward_claimed | BOOLEAN | 성사 신고 여부 |
| reward_claimed_at | TIMESTAMPTZ | 성사 신고 시점 |

## 알려진 한계 (Phase 1)

| 한계 | 설명 |
|------|------|
| 알림 없음 | 신청/수락 시 이메일·앱 알림 없음. 직접 대시보드 확인 필요 |
| 역방향 신청 불가 | 선생님 → 학부모 먼저 연락 불가 |
| 채팅 없음 | 연락처 공개 후 외부 연락 필요 |
| 인증 없음 | 선생님 자격 검증 시스템 없음 |
