# Future Features

> 언젠가 만들 기능. 시기는 미정. roadmap으로 승격되기 전 대기소.

## 🔥 우선순위 높음 (오픈 후 6개월 내 검토)

### 학생 약점 진단 리포트 자동 생성
- 학생 풀이 데이터 → AI 분석 → 학부모 발송용 PDF
- 봉쌤 비전("강사가 수업에만 집중") 직접 구현
- 학부모 상담 자동화 효과

### 시험지 자동 생성 (강사용)
- 조건 입력(학년·난이도·단원·시간) → 시험지 PDF 출제
- 기존 math-hwpx 스킬 활용 가능
- 강사 수업 준비 시간 90% 단축

### 학습 경로 추천 엔진
- 학생 데이터 + 평가원 자료 매칭
- "당신은 다음 5문제를 이 순서로" 권장
- 단순 추천이 아닌 선수학습 위계 기반

### 음성 풀이 해설
- 학생이 막힌 문제 → 음성 질문 → AI 음성 답변
- ElevenLabs + Claude API
- 1:1 과외 느낌 제공
- (오픈 후 안정화 단계에 검토)

- 회원 등급별 차등 공개 (무료=요약, 유료=맞춤·브리핑)

- (오픈 후 6개월~1년 시점)
