> 시장 인식·경쟁 분석. 의사결정의 전제.

## 🎯 핵심 인식 (2026-05-02)

> **"AI 발전으로 사이트·기능 구현 진입장벽 소멸 중.**
> **진짜 모트는 선점·데이터·신뢰."**

### 의미
- 코드 구현은 누구나 → 차별화 자산이 모든 것
- 1인 개발자가 1년 걸리던 일 → 1주일에 끝남
- 진입장벽이 코드 → 아이디어·데이터·신뢰로 이동
- 단, 경쟁자도 똑같이 빨라지므로 **실행 속도가 모든 것**

## 봉쌤이 만들 수 있고 AI가 못 만드는 것

| 자산 | AI 복제 가능? |
|------|---------------|
| 사이트 코드 | ✅ 쉬움 |
| 유사문제 생성 | ✅ 점점 쉬움 |
| 문제은행 (기출) | ⚠️ 중간 |
| **현역 교사 정체성** | ❌ 불가 |
| **평가원 자료 분석 노하우** | ❌ 불가 |
| **누적 학생 학습 데이터** | ❌ 시간 필요 |
| **교사·학원 동업자 네트워크** | ❌ 불가 |
| **MathIQ 브랜드 신뢰** | ❌ 시간+마케팅 |

→ 봉쌤 시간은 위 5가지에 집중.
→ 코드·디자인·반복 작업은 AI에 최대한 위임.

## 선점의 3가지 의미

### 1. 시간적 선점 (Time Moat)
"이 카테고리에 처음 들어간 사람" 인식 확보.
"평가원 분석 교재? MathIQ" 첫인상은 한 번뿐.

### 2. 데이터 선점 (Data Moat)
1년 먼저 시작 = 학생 풀이 데이터 1년치 우위.
합격 예측·약점 진단 정확도가 시간이 지날수록 격차 벌어짐.

### 3. 관계 선점 (Network Moat)
동업자 5명, 학원 10곳, 학부모 1,000명 신뢰 = 1~2년 추격 불가.

## ⚠️ 위협 요소

### 진학사·메가의 견제
- 오픈 후 1년 내 가능성
- 로펌 통한 압박 가능
- **대비**: 출판 전 변호사 자문 1회 (저작권·약관)

### 데이터 정확성 리스크
- 등급컷 오류 = 학생 인생 영향 = 신뢰 즉사
- **대비**: 공식 vs 추정 명확 구분, 면책 조항, 봉쌤 검수

### 크롤링 법적 이슈
- 대학 입학처 PDF 직접 호스팅 ❌
- "원문 링크 + 자체 분석" 형태로
