# 수강료 관리

학생별 수강료 청구 및 입금 확인 기능을 안내합니다.

## 수강료 설정

학생별로 월 수강료를 설정할 수 있습니다.

1. 학생 상세 페이지 진입
2. **수강료 설정** 탭 클릭
3. 월 수강료 금액 입력
4. 결제일(매월 N일) 설정

## 청구서 발송

설정된 결제일에 맞춰 청구서를 발송합니다.

- **수동 발송**: 개별 청구서 직접 발송
- **자동 발송**: 매월 결제일에 자동 발송 (프리미엄 이상)

## 입금 확인

학부모가 입금하면 입금 확인 처리합니다.

- 수동 확인: 청구서에서 [입금 확인] 버튼 클릭
- 자동 확인: 오픈뱅킹 연동 (프로 티어)

## 수강료 이력

월별 수강료 청구/입금 이력을 확인할 수 있습니다.

- 월별 요약 차트
- 미수금 현황
- 엑셀 다운로드
