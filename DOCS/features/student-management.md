# 학생 관리

학생 등록, 수정, 관리 기능을 안내합니다.

## 학생 등록

1. 사이드바에서 **학생 관리** 클릭
2. **[+ 학생 추가]** 버튼 클릭
3. 학생 정보 입력:
   - 이름 (필수)
   - 학년 (필수)
   - 학교
   - 학부모 연락처

## 학생 목록

등록된 학생들의 목록을 확인할 수 있습니다.

- 학년별 필터링
- 이름 검색
- 최근 시험 성적 요약

## 학생 상세

학생 카드를 클릭하면 상세 정보를 확인할 수 있습니다.

- 기본 정보 수정
- 시험 이력 조회
- 성적 추이 그래프
- 학부모 리포트 발송

## 학생 삭제

학생 정보를 삭제하면 관련 시험 기록도 함께 삭제됩니다.

> ⚠️ 삭제된 데이터는 복구할 수 없습니다.
