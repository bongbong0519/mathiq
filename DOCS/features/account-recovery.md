# 계정 탈퇴 및 복구

계정 탈퇴와 복구 절차를 안내합니다.

## 계정 탈퇴

프로필 설정에서 계정 탈퇴를 신청할 수 있습니다.

### 탈퇴 시 처리 사항
- 담당 학생과의 연결이 해제됩니다
- 포인트와 캐시 잔액이 스냅샷으로 저장됩니다
- 30일간 계정이 보관됩니다

### 탈퇴 절차
1. 프로필 > 계정 탈퇴 클릭
2. 탈퇴 사유 입력 (선택)
3. 확인 후 탈퇴 처리

## 계정 복구

탈퇴 후 30일 이내에 복구할 수 있습니다.

### 자동 복구 (50%)
- 탈퇴 후 로그인 시 복구 화면 표시
- [계정 복구하기] 클릭
- 포인트의 **50%**가 자동 반환

### 이의제기 (100%)
- 자동 복구 후 추가 복구 신청 가능
- 운영자 검토 후 나머지 50% 반환
- 복구 후 30일 이내 신청 가능

## 영구 삭제

탈퇴 후 30일이 지나면 계정이 영구 삭제됩니다.

> ⚠️ 영구 삭제 후에는 복구가 불가능합니다.
