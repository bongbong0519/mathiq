# 회계 관리

수입과 지출을 관리하는 기능입니다.

> 💎 이 기능은 **프리미엄** 티어 이상에서 사용 가능합니다.

## 매출 관리

수강료 입금 내역이 자동으로 매출에 반영됩니다.

- 월별 매출 요약
- 학생별 매출 분석
- 전년 대비 비교

## 지출 관리

학원/과외 운영에 필요한 지출을 기록합니다.

### 지출 카테고리
- **임대료**: 학원/교습소 임대료
- **교재비**: 교재 및 학습 자료 구입
- **공과금**: 전기, 수도, 인터넷 등
- **기타**: 그 외 지출

### 지출 등록
1. 회계 관리 메뉴 클릭
2. [+ 지출 등록] 버튼 클릭
3. 날짜, 카테고리, 금액, 메모 입력

## 손익 분석

매출에서 지출을 뺀 순이익을 확인합니다.

- 월별 손익 그래프
- 카테고리별 지출 비중
- 엑셀 리포트 다운로드
