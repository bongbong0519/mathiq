## 비전

> **"전국 인증 수학 선생님 DB를 구축하여,  
> 학부모가 신뢰할 수 있는 과외 선생님을 쉽게 찾을 수 있게 한다."**

MathIQ는 단순 문제은행을 넘어, 수학 교육 생태계 플랫폼으로 발전합니다.

## Phase 3 (1년+, 2027~)

### 선생님 인증 시스템 (DOCS/07-tutor-certification.md 참조)

#### 등급제 (4단계)
| 등급 | 조건 | 배지 |
|------|------|:----:|
| 🥉 브론즈 | 개인과외교습자 신고 또는 학원강사 등록 | - |
| 🥈 실버 | 브론즈 + 학력 증명 | 학력 인증 |
| 🥇 골드 | 실버 + 경력 + 후기 10개(4.5+) | 검증된 선생님 |
| 💎 다이아 | 골드 + MathIQ 실력 테스트 | MathIQ 인증 |

#### 검증 방법
- Phase 1: 봉쌤 수동 검증
- Phase 2: OCR + AI 보조
- Phase 3: 교육청 API 연동 (가능 시)
