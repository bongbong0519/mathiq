# MathIQ 장기 로드맵

> 최종 업데이트: 2026-04-21

---

## 비전

> **"전국 인증 수학 선생님 DB를 구축하여,  
> 학부모가 신뢰할 수 있는 과외 선생님을 쉽게 찾을 수 있게 한다."**

MathIQ는 단순 문제은행을 넘어, 수학 교육 생태계 플랫폼으로 발전합니다.

---

## 현재 완료 기능 (2026-04)

### 문제은행
- ✅ 공공저작물(수능·모평·학평) 문제 수집
- ✅ 과목/단원/유형별 검색
- ✅ Ctrl+V 페이지 검수 시스템
- ⚠️ AI 자동 추출 포기 → 수동 검수 전환

### 시험 관리
- ✅ 시험지 생성 (문제 선택 조합)
- ✅ 학생 응시 / 채점
- ✅ 성적 분석 (문항별, 단원별)
- ✅ 학생 성적 리포트 PDF

### 학생 관리
- ✅ 학생 등록 / 그룹 관리
- ✅ 학부모 연락처 등록
- ⚠️ 학부모 문자 발송 (UI만, 실제 발송 미구현)

### 과외 매칭 (Phase 1)
- ✅ 선생님 프로필 등록
- ✅ 학부모 구인 프로필 등록
- ✅ 선생님 검색 (과목/지역)
- ✅ 매칭 신청 → 수락/거절
- ✅ 연락처 공개 (수락 시)
- ✅ 포인트 차감/리워드

### 포인트 시스템
- ✅ 잔액 관리 (profiles.point_balance)
- ✅ 차감/지급 RPC 함수
- ⚠️ 내역 조회 미구현

## Phase 3 (1년+, 2027~)

### 선생님 인증 시스템

#### 등급제
| 등급 | 조건 | 배지 |
|------|------|:----:|
| 신규 | 가입 후 30일 미만 | 🆕 |
| 일반 | 30일+ 활동 | - |
| 인증 | 자격증 제출 확인 | ✅ |
| 우수 | 인증 + 리뷰 4.5+ | ⭐ |
| 마스터 | 우수 + 매칭 50회+ | 👑 |

#### 인증 가능 자격
- 교원자격증 (정교사, 준교사)
- 학원강사 자격증
- 대학 졸업증명서 (수학/교육 관련)
- 경력증명서

### 리뷰/평점 시스템
- [ ] 학부모 → 선생님 리뷰
- [ ] 평점 (1~5점)
- [ ] 리뷰 신고/관리
- [ ] 가짜 리뷰 방지

### 결제 대행
- [ ] 수업료 결제 대행
- [ ] 에스크로 (수업 완료 후 정산)
- [ ] 수수료 모델 (3~5%)

### 채팅 시스템
- [ ] 인앱 채팅 (Supabase Realtime)
- [ ] 파일/이미지 전송
- [ ] 채팅 내역 보관

### 전국 DB 확장
- [ ] 지역별 선생님 분포 통계
- [ ] 추천 알고리즘 (과목/지역/평점)
- [ ] 학원 협력 프로그램
