# 베타 후 정책 결정 사항

> 작성: 2026-04-27
> 용도: 베타 후 또는 정책 결정 시 처리할 사항 누적

## 가입 분기 정책 (미정)

**비즈니스 의도:**
- 캠퍼스 = "미끼" 티어 (포인트는 많이, 사용은 빡빡)
- 대학생만 캠퍼스 가능
- 일반 사용자는 베이직 시작

**구현 필요:**
1. 가입 폼에 대학생 체크박스 추가
2. 체크 -> campus 배정 + 5,000pt
3. 체크 X -> basic 배정 + 1,000pt
4. 인증 방식 미정 (자기 신고? 학교 이메일? - 베타 후 결정)

**현재 상태:**
- 가입 시 무조건 campus 배정 (위치 A: 7493행, 위치 B: 7787행)
- SIGNUP_BONUS 상수는 이미 정의됨 -> 분기만 추가하면 즉시 작동

## 캠퍼스 제한 빡빡 적용 (미정)

**비즈니스 의도:**
- 캠퍼스가 "미끼" 역할 하려면 사용 제한 빡빡해야 함
- 5,000pt는 받지만 막상 쓸 수 있는 게 적어야 베이직 업그레이드 유도

**제한 검토 항목:**
- 학생수 제한 (현재 3명?) - 더 줄일지
- 학부모 문자 (현재 주 2회) - 유지/축소
- 자료실 접근 (현재 ?) - 검토
- AI 유사문제 - 검토
- 그 외 기능 - 검토

**현재 상태:**
- 제한 항목별로 분산되어 있음
- 정책 결정 후 일괄 적용 필요
