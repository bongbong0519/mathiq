# 첫 시작 가이드

MathIQ를 처음 사용하는 분들을 위한 가이드입니다.

## 선생님 첫 시작

### 1. 프로필 설정
- 사이드바 하단 프로필 버튼 클릭
- 이름, 연락처, 학교/학원 정보 입력
- 프로필 사진 등록 (선택)

### 2. 학생 등록
- 학생 관리 메뉴 클릭
- [+ 학생 추가] 버튼으로 학생 정보 입력
- 학년, 학교, 학부모 연락처 등록

### 3. 시험지 분석
- 시험지 분석 메뉴 클릭
- 시험지 이미지를 업로드하거나 붙여넣기
- AI가 자동으로 문제와 정답 추출

## 학부모 첫 시작

### 1. 자녀 등록
- 프로필에서 자녀 정보 연결
- 담당 선생님 초대 코드 입력

### 2. 과외 선생님 찾기
- 과외 매칭 메뉴에서 조건 검색
- 선생님 프로필 확인 후 매칭 신청
