# 역할 안내

MathIQ는 사용자 역할에 따라 다른 기능을 제공합니다.

## 선생님

개인 과외 선생님 또는 학원 강사를 위한 역할입니다.

- 학생 등록 및 관리
- 시험지 업로드 및 분석
- 성적 리포트 생성
- 학부모 문자 발송
- 과외 프로필 등록

## 원장님

학원/교습소 운영자를 위한 역할입니다.

- 소속 선생님 관리
- 전체 학생 현황 파악
- 매출/수강 현황 대시보드

## 학부모

자녀의 학습을 관리하는 학부모를 위한 역할입니다.

- 자녀 성적 리포트 확인
- 과외 선생님 검색 및 매칭 신청
- 학부모 커뮤니티 참여

## 학생

학습 결과를 직접 확인하는 학생을 위한 역할입니다.

- 본인 성적 리포트 확인
- 학습 진도 그래프
