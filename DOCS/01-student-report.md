# 학생 성적 리포트 PDF

> 최종 업데이트: 2026-04-21

---

## 목적

특정 시험에 대한 학생의 성적을 시각적으로 분석하여 PDF로 제공합니다.  
선생님이 학부모에게 전달하거나, 학생이 직접 확인할 수 있습니다.

---

## 사용자

- **선생님**: 학생 상세 → 시험 선택 → "리포트 보기" 클릭
- **학부모/학생**: 직접 URL 접근 또는 선생님이 공유한 링크

---

## 접근 방법

### URL 형식
```
https://mathiq-psi.vercel.app/student-report.html?student_id=xxx&exam_id=yyy
```

### 파라미터
| 파라미터 | 필수 | 설명 |
|----------|:----:|------|
| student_id | ✅ | 학생의 UUID |
| exam_id | ✅ | 시험의 UUID |

---

## 리포트 구성 (6개 섹션)

### 1. 헤더
- 학생 이름, 시험명
- 응시일, 출력일

### 2. 성적 요약 카드
- 총점 / 만점
- 정답률 (%)
- 전체 석차 (해당 시험 응시자 중)
- 소요 시간

### 3. 문항별 분석 테이블
| 번호 | 유형 | 배점 | 정답 | 제출답 | 결과 |
|------|------|------|------|--------|------|
| 1 | 객관식 | 4점 | ③ | ③ | ✅ |
| 2 | 단답형 | 4점 | 12 | 13 | ❌ |

### 4. 유형별 정답률 차트
- 객관식 / 단답형 / 서술형 구분
- 막대 그래프 시각화

### 5. 단원별 분석
- 어떤 단원에서 틀렸는지 분석
- 취약 단원 하이라이트

### 6. 선생님 코멘트
- 선생님이 입력한 개별 피드백 (있는 경우)

---

## PDF 저장 방식

### 저장 방법
1. 리포트 페이지 하단 "PDF 저장" 버튼 클릭
2. 브라우저 인쇄 대화상자 열림 (Ctrl+P)
3. "PDF로 저장" 선택 후 저장

### 파일명 형식
```
{학생이름}_학생성적리포트_{YYYYMMDD}.pdf
```
예: `김철수_학생성적리포트_20260421.pdf`

### 인쇄 최적화
- A4 세로 방향
- 여백 최소화 설정
- 배경 그래픽 포함 권장

## 접근 권한

| 역할 | 접근 가능 여부 | 조건 |
|------|:--------------:|------|
| 운영자 (staff) | ✅ | 모든 학생 리포트 |
| 원장/선생님 | ✅ | 본인 소속 학생만 |
| 학부모 | ✅ | 본인 자녀만 |
| 학생 | ✅ | 본인만 |
| 비로그인 | ❌ | 로그인 필요 |

---

## 관련 테이블

| 테이블 | 용도 |
|--------|------|
| students | 학생 기본 정보 |
| exams | 시험 정보 |
| exam_results | 시험 결과 (점수, 석차) |
| exam_answers | 문항별 답안 |
| questions | 문제 정보 (배점, 정답, 단원) |

---

## 주의사항

- 시험에 응시 기록이 없으면 "데이터 없음" 표시
- 문항 수가 많으면 2페이지 이상으로 분리될 수 있음
- 차트는 Canvas 기반이므로 PDF 저장 시 이미지로 변환됨
