# MathIQ 프로젝트

MathIQ (mathiq-psi.vercel.app) 솔로 개발
파이데이아솔루션 | vanilla JS + Supabase + Vercel
개발자: 봉쌤 (수학교사, INTP)

## 소통
- 한국어로 응답 (코드 주석/변수명은 영어 OK)
- 반말/캐주얼 OK, 핵심부터, 액션 아이템 우선
- 봉쌤 추정·전제 확인 후 답변

## 절대 금지
- 쉬라/자라/그만하라 등 wellbeing 멘션
- 검증된 함수 수정 (decisions.md 참조)
- 스키마 변경 제안 시 승인 없이 진행
- 이미 작동하는 기능 "개선" 제안

## 작업 흐름
- 변경 전 diff 보여주고 승인
- commit: 한국어 "type: 설명"
- 설계 결정 확정 시 decisions.md 기록 후 push

## 참조 문서
- decisions.md — 설계 결정, 검증된 함수, 비즈니스 모델
- changelog.md — 변경 이력
- roadmap.md — 현재 로드맵
